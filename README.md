# Product_Trading 校园二手交易平台


1. 简述：一个web端的校园二手交易平台
2. 使用语言：java、jsp、sql、html、css、jquery
3. 开发工具：IDEA
4. 开发环境：jdk1.8
5. 数据库：MySQL8.0.26

## 已实现功能
1. 登录注册
2. 分页
3. 首页模糊查询
4. 购物车、购物车结算生成订单
5. 发布商品
6. “我发布的”可在“我的”页面点击查看，并进行修改删除操作
